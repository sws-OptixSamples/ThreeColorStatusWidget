# Three-Color Status Widget

StatusColor output is changes based on the StatusValue input.  
Interpolation between colors will be configured via the Min, Mid, and Max properties.  
Configure the Min/Mid/Max colors  
Configure the Min/Mid/Max values  
Link StatusValue to a variable source  
Change the variable source and see the StatusColor output change
